# First-timers Guide

Welcome new contributors! Steps to get started:
1. Read CONTRIBUTING.md and CODE_OF_CONDUCT.md.
2. Look for issues labeled `good-first-issue`.
3. Comment on an issue to say you're working on it.
4. Fork the repo, create a branch, implement, add tests, run `npm test`.
5. Open a PR and reference the issue.

Tips
- Ask questions on the issue if uncertain.
- Keep PRs small and focused.
