## 2024-05-28 - Add Initial Loading State to Popup
**Learning:** For browser extension popups fetching data from IndexedDB/external APIs upon load, failing to show an initial loading state results in a broken/stuck visual experience while the async operations run. The UI should instantly inject a loading state into the container, and `aria-live="polite"` handles screen reader announcements seamlessly during the brief fetch pause.
**Action:** Always inject an explicit loading indicator (e.g., text or spinner) into dynamic data containers before initiating complex async chains like API calls or IndexedDB lookups. Ensure `aria-live` is set directly on the injected state.
