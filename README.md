# Manual Installation of a Downloaded Thunderbird Extension

## 1. Open Thunderbird Add-on Manager

- **Open Thunderbird**: Start your Thunderbird application.
- **Open Application Menu**: Click on the menu icon (three horizontal lines) in the upper right corner.
- **Select "Add-ons and Themes"**: Choose this option from the menu that appears.

## 2. Prepare Installation

- **Click on the Gear/Settings Icon**: This is located at the top right of the Add-on Manager tab.
- **Select "Install Add-on From File..."**: A file browser window will open.

## 3. Select and Install the Extension

- **Find the `*.xpi` file**: Navigate to the location where you saved the extension file.
- **Select File**: Click on the `*.xpi` file to select it.
- **Click "Open"**: This will start the installation process.

## 4. Confirm Installation

- **Check Permissions**: Thunderbird will display the requested permissions for the extension. Review these carefully.
- **Click "Add" (or "Install")**: Confirm the installation with the corresponding button.

## 5. Restart Thunderbird

- **Restart Required**: Thunderbird may prompt you to restart the program.
- **Click "Restart now" or restart manually**: This will apply the changes.

## Troubleshooting

- **Check Compatibility**: Ensure the extension is compatible with your Thunderbird version. You can find this information in the extension's description or on its website.
- **Exclude File Corruption**: If errors occur, re-download the `*.xpi` file to rule out a corrupted file.
