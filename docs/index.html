Über das Projekt

Thunderbird Security Antivirus ("Thundy AV") prüft E-Mail-Anhänge automatisch gegen über 70 Virenscanner und Sandbox-Systeme. Es nutzt die kostenlose API von Hybrid Analysis, um Bedrohungen zu erkennen, die herkömmliche Virenscanner oft übersehen.

Einrichtung & API Key

Um die Erweiterung nutzen zu können, benötigen Sie einen kostenlosen API-Schlüssel. Da dies ein Community-Projekt ist, nutzt jeder Benutzer seinen eigenen Zugang (Fair Use).

Schritt 1: Registrieren Sie sich kostenlos auf hybrid-analysis.com.
Schritt 2: Gehen Sie nach der Anmeldung auf Ihr Profil (oben rechts) -> "Profile".
Schritt 3: Kopieren Sie den "API Key" unter API Settings.
Schritt 4: Öffnen Sie in Thunderbird den Add-on Manager, klicken Sie auf das Zahnrad bei "Thundy AV" und fügen Sie den Schlüssel ein.

Datenschutz & Sicherheit

Transparenz ist uns wichtig:

  • Die Erweiterung liest keine E-Mail-Inhalte, nur Anhänge.
  • Dateien werden direkt von Ihrem Computer an hybrid-analysis.com gesendet.
  • Der Entwickler dieses Add-ons hat keinen Zugriff auf Ihre Dateien oder Ihren API-Schlüssel.
  • Der Quellcode ist offen (Open Source) und kann hier auf GitHub überprüft werden.
