About the Project

Thunderbird Security Antivirus ("Thundy AV") automatically checks email attachments against over 70 virus scanners and sandbox systems. It uses the free API of Hybrid Analysis to detect threats that traditional virus scanners often miss.

Setup & API Key

To use the extension, you need a free API key. As this is a community project, each user utilizes their own account (Fair Use).

Step 1: Register for free at hybrid-analysis.com.
Step 2: After logging in, go to your profile (top right) -> "Profile".
Step 3: Copy the "API Key" under API Settings.
Step 4: In Thunderbird, open the Add-on Manager, click on the gear icon next to "Thundy AV", and paste the key.

Privacy & Security

Transparency is important to us:

  • The extension reads no email content, only attachments.
  • Files are sent directly from your computer to hybrid-analysis.com.
  • The developer of this add-on has no access to your files or your API key.
  • The source code is open (Open Source) and can be reviewed here on GitHub.
